# Compiled Externals

Precompiled graf externals for people who don't want to build from source.

Currently available: macOS, Apple Silicon only. Intel Mac and Windows builds are not provided yet.

Installation: download graf.zip and uncompress it directly into your Max 8 Packages folder (~/Documents/Max 8/Packages/). This creates the graf package with all externals ready to use — no further setup needed.

If you're on an unsupported platform, or want the latest unreleased changes, build from source instead (see the main README).
